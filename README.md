# iOS
## Requirements
- iOS 9.0+
- Xcode 7.2+

## Setup

- `$ cd ReceptionApp/iOS`

- `$ pod install`

- `$ git submodule update --init --recursive`

- set the following items in `Configuration.swift`
	- Company Name
	- Company Logo
	- API Token
	- Sent Sound
	- Floor
	- Color
	- Font
